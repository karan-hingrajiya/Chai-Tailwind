Documentation

Get Started in 30 Seconds

1

Add the script tag

Drop one script tag into your HTML head. No npm install, no terminal commands, no config file needed.

<script src="chai.js"></script>
2

Write chai- classes

Use the prefix chai- before any utility name. The engine scans your DOM and generates CSS automatically.

<div class="chai-flex chai-p-5 chai-bg-purple chai-rounded-lg">
  Hello World
</div>
3

Open in browser

That's it. Your styles are generated and injected at runtime. No build step, no waiting, no output file.

Class Name Syntax

Spacing

Format: chai-{prop}-{token}

chai-p-5 padding: 20px
chai-mt-3 margin-top: 12px
chai-gap-4 gap: 16px

Colors

Format: chai-bg-{color}-{shade}

chai-bg-blue default blue
chai-bg-red-500 red shade 500
chai-textcol-green green text

Static

Format: chai-{utility}

chai-flex display: flex
chai-flex-col flex-direction: column
chai-hidden display: none

How The Engine Works Internally

01

Scanner reads the DOM

document.querySelectorAll("*") — every single element on the page is visited. Each element's classList is checked for the chai- prefix.

02

Parser splits the class name

"chai-bg-red-500" is split into parts. The engine tries 3-word props first, then 2-word, then single-word. This is what makes chai-flex-col and chai-items-center work correctly.

03

Token resolver looks up the value

Each property knows which map to use — spacing map for padding/margin, colors map for backgrounds and text. Shade values like 500 go one level deeper into the color object.

04

CSS injected into style tag

A single <style> tag is created in <head>. Each generated rule is appended once — a Set tracks what's already generated so nothing is duplicated.
